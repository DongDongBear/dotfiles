# 动动的 dotfiles

## 包含

- **Ghostty** — Catppuccin Mocha 毛玻璃主题 + Maple Mono 字体 + 分屏快捷键
- **Neovim** — 配合 Cursor VSCode Neovim 插件，flash.nvim 跳转 + 自定义键位
- **Cursor** — keybindings（jj 退出插入模式等）

## 安装

```bash
git clone https://github.com/dongdong/dotfiles.git ~/dotfiles
cd ~/dotfiles
bash install.sh
```

## 依赖

```bash
brew install --cask ghostty
brew install neovim
# Cursor 里安装 VSCode Neovim 插件: asvetliakov.vscode-neovim
# 字体: Maple Mono NF CN
```

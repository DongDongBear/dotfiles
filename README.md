# 动动的 dotfiles

## 包含

- **Ghostty** — Catppuccin Mocha 毛玻璃主题 + Maple Mono 字体 + 分屏快捷键
- **Neovim** — 配合 VSCode Neovim 插件，flash.nvim 跳转 + 自定义键位
- **VSCode 系** — 通用 keybindings，适配 Cursor / VS Code / Windsurf / VSCodium

## 安装

```bash
git clone https://github.com/DongDongBear/dotfiles.git ~/dotfiles
cd ~/dotfiles
bash install.sh
```

## 依赖

```bash
brew install --cask ghostty
brew install neovim
# 编辑器里安装 VSCode Neovim 插件: asvetliakov.vscode-neovim
# 字体: Maple Mono NF CN
```
